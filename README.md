# Blocky-OS v7.6 

A powerful WiFi security and analysis tool built for **ESP8266** and **ST7735 1.8" TFT Display**.

##  Features
- **Scanner Pro:** Advanced network scanning and RSSI monitoring.
- **WiFi Killer:** Multi-channel jamming and deauth attacks.
- **Beacon Engine:** Spam fake access points across the spectrum.
- **Evil Portal:** Capture connections with a captive portal.
- **Rickroll HQ:** Troll everyone with legendary SSID beacons.

##  Hardware Required
- ESP8266 (NodeMCU or Wemos D1 Mini)
- ST7735 1.8" TFT Display
- Custom 8-button matrix keyboard
